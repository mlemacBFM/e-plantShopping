# e-plantShopping

**e-plantShopping** is a React-based e-commerce web application for browsing and purchasing plants online.

## Overview

This project provides a modern plant shopping experience with the following features:

- Browse a curated catalog of plants organized by category
- Add plants to a shopping cart
- View and manage cart items with quantity controls
- See real-time cart totals
- Responsive UI built with React and Vite

## Tech Stack

- **React** — component-based UI
- **Redux Toolkit** — global cart state management
- **Vite** — fast development server and build tool

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
